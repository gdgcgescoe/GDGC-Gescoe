## ğŸš€ Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository**
    
   ```bash
   For Frontend 
   git clone https://github.com/gdgcgescoe/GDGC-Gescoe.git
   cd GDGC-Gescoe/frontend
   ```


